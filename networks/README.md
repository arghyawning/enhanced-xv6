# Networks

## Part A

### Basic

#### TCP

Directory: `./partA/basic/tcp`

Run the server code:
```bash
gcc -o server server.c
./server
```

Run the client code:
```bash
gcc -o client client.c
./client
```

#### UDP

Directory: `./partA/basic/udp`

Run the server code:
```bash
gcc -o server server.c
./server <port number>
```

Run the client code:
```bash
gcc -o client client.c
./client <port number>
```
