# Networks

## Part A

### Basic

#### TCP

Run the server code:
```bash
gcc -o server server.c
./server
```

Run the client code:
```bash
gcc -o client client.c
./client
```

#### UDP

Run the server code:
```bash
gcc -o server server.c
./server <port number>
```

Run the client code:
```bash
gcc -o client client.c
./client <port number>
```
